# Zocial 1.2.0

*Released 2026-06-08*

A focused polish release: a refreshed landing page, footer and About page, plus a handy "scroll to top" button for conversations.

## ✨ New features

- Floating "scroll to top" button in threads: When you open a conversation (e.g. via a reply or boost) you often land in the middle of a thread. A floating button now appears once you've scrolled past the first post and smoothly takes you back to the top of the conversation. It's styled like the home compose button (theme-aware), shows an up-chevron, sits top-right, and only appears in thread views.

## 💅 Redesigned landing page

- Modern, centered layout: the app logo (with a subtle glow) on top, a bold "Zocial" title, the tagline, and the login button — all stacked and centered.
- New tagline: "A minimalistic opinionated Mastodon web client." (localised for German, Spanish, French and Russian).
- Static app name: the landing page now always shows "Zocial" instead of falling back to the host name on self-hosted/dev builds.

## 🧾 Reworked footer

- New footer, centered: Zocial v<version> · Powered by Zeitfresser · Donations · Privacy Policy.
- The version number is always current (read live from the build).
- "Donations" and "Privacy Policy" link into the About page; "Powered by Zeitfresser" opens in a new tab. The footer text is kept in English across all languages for consistency.

## 💛 About page

- New "Donations" section explaining that Zocial is a non-profit project that's free to use, with optional donations to help cover the public instance's running costs (domain & hosting) — including a PayPal donation link. Translated into every supported language.
- All About pages aligned with the English version and brought up to date (de/es/fr/ru): correct "fork of Enafore" wording, Docker/nginx hosting note, translation-provider note, Version section — removing stale leftovers.

## 🔧 Maintenance

- Deploy URLs (OAuth app website, canonical URL) now point to zocial.social; the migrate page text was corrected (the Git repository stays at git.ztfr.eu).
- README updates.
